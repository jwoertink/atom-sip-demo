# A demo app with atom-shell
This demo will use sip.js for WebRTC

## Install
* clone repo
* cd into directory
* `npm install`
* `grunt`
* `./build/Atom.app/Contents/MacOS/Atom .`

## Setup
[Setting up an Asterisk Server](https://github.com/jwoertink/webrtc-demo)

## Notes
